# Install RippleUI in your project

How to install Ripple UI in your project

## Setup

You can install RippleUI v4 Fork using npm, yarn or CDN.

1. To install the plugin, first install [Tailwind CSS v4](https://tailwindcss.com/docs/installation).

```bash disableTabs={true}
npm install @wisdom_plus/namiui
```

2. Add plugin to your `tailwind.config.js` file

```js disableTabs={true}
module.exports = {
  // ... the rest of your config
  plugins: [require("@wisdom_plus/namiui")],
};
```

3. Ensure your CSS entry imports Tailwind:

```css disableTabs={true}
@import "tailwindcss";
```

4. CDN

If you prefer to use CDN, you can add the following link to your HTML file, but consider you can't remove the unused styles.

<CDN/>


Now you are set to use RippleUI components in your project.

```html
<button className="btn btn-primary">Hello World!</button>
```
