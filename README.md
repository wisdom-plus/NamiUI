Tailwind CSS v4-ready fork of RippleUI.

## Usage

```bash
npm install namiui
```

Add the plugin to your `tailwind.config.js`:

```js
module.exports = {
  // ...rest of your config
  plugins: [require("namiui")],
};
```

Import Tailwind in your CSS entry:

```css
@import "tailwindcss";
```

Use classes in markup:

```html
<button class="btn btn-primary">Click me!</button>
```
